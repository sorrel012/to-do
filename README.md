# 📝 To Do List
### 일정을 관리할 수 있는 웹사이트 <br><br>

  
## 🖥 주요 기능
- 카테고리 관리
  - 원하는 카테고리를 추가할 수 있다.
  - 카테고리별 일정을 확인할 수 있다.

- 일정 관리
  - 선택된 카테고리에 일정을 추가할 수 있다.
  - 일정의 카테고리를 변경할 수 있다.
  - 선택된 카테고리에서 일정을 삭제할 수 있다.

<br>
